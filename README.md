# Como executar o projeto

## Front end web
Pré-requisitos: npm / yarn

```bash
# clonar repositório
git clone https://github.com/DaviAugustoMS/github_finder.git

# entrar na pasta do projeto front end web
cd github_finder

# instalar dependências
yarn install

# executar o projeto
yarn start
```
